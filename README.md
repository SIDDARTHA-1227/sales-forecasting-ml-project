# 📈 Sales Forecasting Using Machine Learning & Time Series

## 📌 Project Overview
This project builds an end-to-end sales forecasting system using retail transaction data.

The workflow includes:
- Data Cleaning
- Feature Engineering
- Exploratory Data Analysis
- Machine Learning Modeling
- Time-Series Forecasting (ARIMA)
- Model Comparison
- Future 30-Day Forecast

## 🛠 Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn
- Statsmodels (ARIMA)
- Google Colab

## 📊 Models Implemented
- Linear Regression
- Random Forest
- ARIMA (Best Performing Model)

## 🏆 Results
ARIMA achieved the lowest forecasting error compared to other models.
